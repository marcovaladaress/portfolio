# Skill: landing-page-portfolio — Design Spec

**Data:** 2026-05-11  
**Status:** Aprovado

## Propósito

Skill de diretrizes para criação de landing pages de portfólio/freelancer em Next.js + Tailwind. O agente segue um checklist de seções obrigatórias com orientações de copy, hierarquia visual e padrões de Tailwind.

## Gatilho

Usar quando o agente for criar ou revisar uma landing page de portfólio ou freelancer com Next.js + Tailwind.

## Abordagem escolhida

**Checklist de seções** — estrutura fixa de 7 seções com diretrizes específicas por seção.

## Estrutura de seções

| # | Seção | Objetivo |
|---|-------|----------|
| 1 | Hero | Primeira impressão — quem é, o que faz, CTA principal |
| 2 | Sobre | Credibilidade e personalidade |
| 3 | Projetos | Prova de trabalho com contexto |
| 4 | Stack / Habilidades | Sinalização técnica |
| 5 | Depoimentos | Prova social |
| 6 | CTA final | Facilitar contato e conversão |
| 7 | Rodapé | Links e créditos |

## Formato por seção

Cada seção terá: objetivo, o que incluir, o que evitar, dicas de Tailwind.

## Conteúdo adicional

- Princípios globais (mobile-first, CTA único, hierarquia visual)
- Erros comuns
- Checklist final de revisão
